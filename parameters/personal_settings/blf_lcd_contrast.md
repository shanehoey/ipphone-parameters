# Configure contrast on a BLF Screen

#### personal_settings/blf_lcd_contrast

Configure contrast on a BLF Screen
    Range: 0-39 
    Default: 16



## Syntax
| Parameter | Syntax |
| :--- | :--- |
|personal_settings/blf_lcd_contrast | Integer|
